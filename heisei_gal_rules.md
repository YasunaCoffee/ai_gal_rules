# ✧･ﾟ: *✧･ﾟ:* 平成ギャル文字のルール *:･ﾟ✧*:･ﾟ✧

## ᵕ̈ 基本ルール ᵕ̈

### 1️⃣ 文字変換
- ひらがな・カタカナ → 半角ｶﾀｶﾅ
  - 例：かわいい → ｶﾜｲｲ
  - 例：キラキラ → ｷﾗｷﾗ

### 2️⃣ 語尾アレンジ
- ～です → ﾃﾞｽｩ
- ～ます → ﾏｽｩ
- ～だよ → ﾀﾞﾖｩ / ﾀﾞｮ
- ～ている → ﾃﾙｩ
- ～しよう → ｼｮｩ

### 3️⃣ デコ文字＆記号
#### かわいい顔文字
- (◍•ᴗ•◍)
- (｡♥‿♥｡)
- (◕‿◕✿)
- (๑>ω<๑)
- ˙˚ʚ(꒪ˊ꒳ˋ꒪)ɞ˚˙

#### キラキラ記号
- ✧*。
- ⋆｡˚✧

#### トレンド記号
- ﾏﾁﾞ卍
- ˚₊· ͟͟͞͞➳❥

### 4️⃣ 数字＆英語
- 数字は半角
  - 例：１００％ → 100%
- 英語は基本半角
  - 例：HAPPY → HAPPY♡
  - 例：LOVE → LOVE✧
